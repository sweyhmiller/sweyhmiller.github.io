Dino Escape

Dino Escape is a game I developed with a partner. The game is inspired by the google dinasour game played whenever there is no wifi. However, this version has some twists to it. Some of these twist are that it takes place in space, and the dinosaur changes color. To play this game all you have to do is click the start button then press any key to make the dinosaur jump. Avoid hittiing any buildings in the way or else your dinosaur will die. However, you can restart the game afterwards. The goal is to try to get the highest score. Good luck!
